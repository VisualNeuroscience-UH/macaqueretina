# MacaqueRetina

Python software to build a model of the macaque retina and convert various visual stimuli to ganglion cell action potentials. 

The simulator can be run either directly or imported as package.  

When run directly, all parameters are read from yaml files at the macaqueretina/parameters folder. 

When imported all parameters are first read from the yaml files, but you have the option of changing parameters at runtime.

Please cite this software as: Vanni S, Hasana S, Vedele F, Hokkanen H: Macaque Retina Simulator (in preparation)
